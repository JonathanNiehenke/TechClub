What is Python

  • Interpreted
  • Multi-paradigm
  • Dynamically typed
  • By Guido van Rossum
  • "Python is an interpreted, multi-paradigm, dynamically typed programming language written by Guido van Rossum and first released in February 1991."
  • Today, supported and developed is through the Python Software Foundation.
  • Interpreted means it is executed by a program rather than by the system directly
  • Like Java (another programming language), Python compiles the source code before interpretation
  • Multi-paradigm means it supports many styles of programming.
  • Imperial where instructions are given in execution order
  • Procedural where the imperial steps are grouped up and later call to run
  • Object Oriented Programming where procedures are exclusively associated with data
  • Dynamically typed means the variable's data type is inferred rather than declared

Generators

  • Efficient
  • Function
  • Expression
  • Generators - yield individual elements as it goes rather it all at once.
  • Generator function - like a function but a generator:
  • Defined like a function - "def genfunctionName(perameters):"
  • But yields rather than returns - "yield commaSeparatedValues"
  • Called like a function - "functionName(arguments)"
  • Comprehension/generator expressions - highly beneficial syntactical sugar:
  • Expressed with delimiters - (): generator, []: list, {}: set.
  • (returnedExpression for Element in Container if Condition)
  • Dictionary comprehension is expressed like:
  • {keyExpression: valueExpression for value1 in container}
